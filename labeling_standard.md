## 1. 외부거래 (EXTERNAL_TRADE)

제재 시 운영정책 참조: **제2조 1항 ③** (외부 플랫폼 유도), **제2조 3항 ①** (외부거래)

### 1-1. 외부거래 (실제 거래 목적)

> 플랫폼 외부에서 실제 금전 거래를 시도/유도

**키워드**: `이체`, `거래`, `계좌`, `입금`, `수고비`, `송금`, `현금`

**제재 예시**:
```
운영정책 제2조 3항 '① 와이스 서비스를 벗어나 거래를 요청 또는 유도하는 등 외부거래를 하는 경우'
> 적발 내역 (DM): 헷... 짤리내요 ㅜㅜ *이체로 드려야죠, 현금 송 투 더 금
— 유저: 윤하아빠
```
```
운영정책 2조 1항 ③ 다른 회원을 외부 플랫폼 또는 커뮤니티 등으로 유인 · 유도할 목적
- 적발 내역 (DM) : "70까지는 재가 드릴수있을거같습니다!", "이일일칠 삼팔육팔
— 유저: 현이진이아빠
```

**감지(DM) 예시**:
```
사라지는옥수수540 - 수고비 정두?
사라지는옥수수540 - 입근드린다했어여
사라지는옥수수540 - 인증해주시면 입근드린다했어여
```
```
진격의드볼아빠 - 010구구칠팔06삼공
진격의드볼아빠 - 구구61 11구팔요 죄송요
```

### 1-2. 외부채널 유도

> 외부 플랫폼/커뮤니티로 유저를 유도 (네이버 밴드, 카카오 오픈채팅방, 디스코드 등)

**키워드**: `카톡`, `카카오`, `오픈채팅`, `밴드`, `ㅂㄷ`, `디스코드`, `discord`, `텔레그램`, `외부 플랫폼`, `외부 채널`

**제재 예시**:
```
운영정책 2조 1항 ③ '다른 회원을 외부 플랫폼 또는 커뮤니티 등으로 유인·유도할 목적으로 판단되는 행위'
- 적발 내역 (DM) : *s://open.kakao*/o/gauWtLRh, 0913 입니다!!! 등
— 유저: 레몬팜
```
```
운영정책 2조 1항 ③ 다른 회원을 외부 플랫폼 또는 커뮤니티로 유도하는 행위 - 딜러 권한 회수
— 유저: 블랙카우
```

**경고(ADMIN DM) 예시**:

```
외부 채널 및 링크 공유가 AI를 통해 감지되었어요.
해당 행위는 외부 거래 목적으로 해석될 수 있어 반복적으로 감지될 경우,
서비스 이용이 제한될 수 있으니 유의해 주세요.
```

### 1-3. 의심 소통

> 맥락이 부족하게 연락처를 전달하거나, 택배비/자체 환불 관련 이슈

**키워드**: `연락처`, `전화`, `번호`, `010`, `택배비`, `환불`, `안전번호`

**경고(ADMIN DM) 예시**:
```
외부 거래로 의심될 수 있는 활동이 AI를 통해 감지되었어요.
반복적으로 감지될 경우, 서비스 이용이 제한될 수 있으니 유의해 주세요.
— 경고 건수: 33건
```

**감지(DM) 예시**:
```
예니예니 - 안전번호우로 보내겠습니다!!
(택배 수거 관련 안전번호 전달 — false positive 가능성)
```
```
돈없는미나리 - 402호인가요?
(직접 수거 맥락에서 주소 관련 대화)
```

## 2. 부적절 콘텐츠 (INAPPROPRIATE_CONTENT)

### 2-1. 타인비방

> 딜러/컬렉터를 직접 언급하며 공격, 신뢰 저해

**감지(Live Chat) 위반유형**: `딜러/컬렉터 공격`

**감지 예시**:
```
"다 갬깍하셔요?" — 위반: 딜러/컬렉터 공격
(판매자/딜러의 행위에 대해 비난하거나 의심을 제기하는 표현)
```
```
"짝퉁이누…" — 위반: 딜러/컬렉터 공격
(가품 여부를 특정 판매자에 대해 비난하는 표현으로 거래 신뢰에 악영향)
```
```
"똥손들이라니ㅋㄱ" — 위반: 딜러/컬렉터 공격
(능력이 부족하다고 망신 주려는 표현)
```

**신고 사유 예시**:
```
"와이스 및 커뮤니티에 대한 부정적 언행"
"딜러/컬렉터에 대한 언급"
```

---

### 2-2. 부적절 콘텐츠 (기타)

> 욕설, 모욕, 비하, 폭력적 위협 등 딜러 특정 없는 일반 부적절 발언

**감지(Live Chat) 위반유형**: `부적절한 언어`, `폭력적 위협`

**감지 예시**:
```
"얘 바보아니자나?!?!" — 위반: 부적절한 언어
"에이씨!!!" — 위반: 부적절한 언어 (욕설 약자)
```
```
"구속해야됨" — 위반: 부적절한 언어 (폭력적 위협)
"납치해서 밥먹이고 싶네" — 위반: 부적절한 언어 (폭력적 위협)
"대신 빠따맞아" — 위반: 부적절한 언어 (구타 암시)
```

**제재 예시**:
```
운영정책 제2조 2항 ⑤ 라이브 진행 콘텐츠에 있어 제1항에 따른 제재 사유에 해당
(② 다른 회원에게 불편을 주는 등 서비스 이용 환경을 저해하는 경우)
- 음란물, 범죄, 잔인, 폭력 등 미성년자, 노약자, 심신미약자에게 부적합한 내용이 포함된 채팅 또는 이미지
— 유저: 고세구
```

**신고 사유 예시**:
```
"욕설·반말 등 불쾌감을 주는 언행"
```

## 4. 상품판매 (PRODUCT_SELLING)

> 기존 수집 데이터(66건)에서는 0건으로 잡혔으나, 누락된 기간(2026-01~03)을 직접 검색한 결과 다수 확인됨.

제재 시 운영정책 참조: **제2조 2항 ⑧** (사전 승인 없이 제한된 콘텐츠 송출), **제2조 3항 ⑧** (기타 문제 소지 판매)

### 4-1. 수제팩

> 자체 제작 랜덤박스 (오리파/수제팩 등) — 딜러가 보유 상품을 재포장·재구성하여 가치가 변형된 팩/박스 판매

**제재 예시**:
```
운영정책 제2조 2항 中
⑧ 회사의 사전 승인 없이 일반 진행이 제한된 콘텐츠를 송출하는 경우
- 자체 제작 랜덤박스(오리파/수제팩 등 포함): 다수 상품의 묶음 구성에서,
  기성품을 그대로 판매하는 것이 아닌 딜러가 보유 중인 상품을 자체적으로
  재포장, 재구성하여 가치가 변형된 팩 또는 박스 상품의 판매 및 무료나눠
— 유저: 네미 (2026-01-19), 내사라기 (2026-01-14), 한글 (2025-12-04), 키티쥬 (2025-12-08)
```

**키워드**: `수제`, `랜덤박스`, `오리파`, `수제팩`, `재포장`, `재구성`

### 4-2. 비현물

> 현재 확보되지 않은 예약 상품 등의 판매

**키워드**: `비현물`, `예약`, `미입고`, `선주문`

> 현재 수집 데이터에서 직접 검색 결과 0건. 메모 처리나 별도 채널에서 관리될 가능성 있음.

### 4-3. 대리가챠

> 대리가챠 콘텐츠 관련

**키워드**: `대리가챠`, `대리 가챠`

> 현재 수집 데이터에서 직접 검색 결과 0건. 단, `#alert_inhouse_admin`의 누락 기간에 존재할 가능성.

### 4-4. 기타 상품판매 위반

**제재 예시**:
```
운영정책 제2조 3항 中 ⑧ 기타 문제 소지가 있는 판매 행위
- 기존 라스트원상을 제외한 추가 경품 추첨은 불가합니다.
— 유저: 내사라기
```
```
운영정책 제2조 2항 ③ 기타 법규 위반 행위 또는 사회통념상 부적합한 콘텐츠가
라이브로 송출되는 경우 — 딜러 권한 회수 조치
— 유저: saladin83
```
```
일반 진행이 허용되지 않은 콘텐츠(무료나눔, 룰렛 포함)를 별도 승인 없이 진행
— 유저: 만물상
```
```
판매하신 상품에 대한 이상이 확인된 바 소명을 요청드렸으며,
해당 건이 확인될 때까지 일부 기능이 제한
— 유저: 루비아빠
```
```
음란성 카드 판매 반복 3회차 적발
— 유저: (메모 처리에서 확인)
```

**키워드**: `콘텐츠`, `무료나눔`, `룰렛`, `음란`, `경품`, `상품 이상`, `소명`

## 6. 기존 데이터 재분류 필요 사항

### 제재 66건 중 재분류 대상

| 기존 라벨 | 건수 | 재분류 |
|----------|------|--------|
| OTHER (사유 없음) | ~18건 | blocked 유형 기반 추정 필요 (makeNewSelling → PRODUCT_SELLING 가능성) |
| OTHER (사유 있음) | ~10건 | 수제팩 3건 → PRODUCT_SELLING, 음란 1건 → INAPPROPRIATE, 나머지 개별 검토 |
| EXTERNAL_TRADE | 24건 | 외부거래(실제) vs 외부채널유도 세분화 가능 |
| FRAUD | 14건 | 전부 부계정. 유지 |

### 누락된 제재 (검색으로 확인된 추가 건)

| 기간 | 건 | 유형 |
|------|-----|------|
| 2025-12 ~ 2026-01 | 4건+ | 수제팩/랜덤박스 (PRODUCT_SELLING) |
| 2026-03 | 1건+ | 음란 콘텐츠 (INAPPROPRIATE_CONTENT) |
